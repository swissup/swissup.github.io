# Configuration

Argento Hyvä configuration is located at
_Stores > Configuration > Swissup > Argento Hyvä Chic_.

![General section](argento_hyva_1.png)

### Layered Navigation

Option | Description
-------|------------
Expand All Filters by Default | If set to Yes, all filter options will be open by default.

### Product Page

Option | Description
-------|------------
Show More | If set to 'Yes', 'Show more' will be used for the short description.
