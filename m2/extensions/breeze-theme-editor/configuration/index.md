# Configuration

Navigate to **Stores > Configuration > Swissup > Breeze Theme Editor**.

## General Settings

Option | Description
-------|------------
**Enable Theme Editor** | Enable or disable Breeze Theme Editor. When disabled, the editor panel will not be accessible from the frontend.

## Accessing the Theme Editor

Once enabled, a button **"Open Frontend in Theme Editor Mode"** appears on the same configuration page.

Click it to open the storefront with the editor panel loaded. The URL contains a time-limited access token (valid for **3 hours**).

You can also copy the token URL manually and share it with a colleague — no admin credentials required to use the editor once you have the link.

> **Note**: Access tokens are regenerated automatically each time you visit the configuration page.
