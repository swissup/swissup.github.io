# Checkout Consumer Rights

Checkout Consumer Rights provides following features:

 -  Ability to show Legal Guarantee Notice at the product and checkout pages.
 -  Ability to register order withdrawal request from `sales/guest/view` page
    for both registered and guest customers.

This module is bundled into [Firecheckout extension](/m2/extensions/firecheckout/),
but can be installed separately if needed.
