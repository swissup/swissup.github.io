# Breeze Content Builder

Visual drag-and-drop editor for building CMS pages and adding content to product and category pages — without writing any code.

## Modules

Breeze Content Builder is a set of modules shipped as part of [Breeze Enterprise](https://breezefront.com/docs/installation) and a separate package:

| Module | Purpose |
|---|---|
| **Breeze Content Builder** | Edit CMS pages with a drag-and-drop component editor |
| **Breeze Layout Builder** | Add content blocks to product and category pages |
| **Breeze Content Builder Migration** | Import existing Page Builder content into the new editor |
